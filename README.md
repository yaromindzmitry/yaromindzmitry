## About

I work across the full QA cycle — from writing automated test suites and running load scenarios to identifying security vulnerabilities and delivering structured client reports. ## What I Do

### ⚡ Performance Testing
- **k6 Browser Mode** load testing across multi-locale production environments (15+ domains)
- Custom threshold configuration: TTFB < 700ms, Page Load < 8s
- Grafana Cloud integration for real-time metrics dashboards
- Structured SLOW_REPORT log parsing and bottleneck analysis
- Comparative locale performance reporting (Word/PDF deliverables)

### 🔐 Security Testing (Black-Box)
- Authorized penetration testing on commercial web platforms
- OWASP ZAP scanning with custom severity remapping (Trivial / Minor / Major / Critical)
- Findings: Reflected XSS, rate limiting gaps, PII exposure in cookies, path disclosure, API info leakage
- Reports mapped to OWASP Top 10, GDPR, 152-ФЗ
- Color-coded Excel + formal Word report delivery

### 🤖 Test Automation
- **Selenium + Pytest** UI automation with Page Object Model
- CI/CD integration via GitHub Actions
- Allure reporting, Docker-based test environments
- Cross-platform: Linux / macOS

### 🛠️ Bots & Tooling
- Telegram bot development (Python, aiogram)
- CRM integrations (Google Sheets API)
- Prometheus + Grafana monitoring for bots
